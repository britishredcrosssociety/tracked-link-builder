Tracked Link Builder for BRC

This tool creates links that are tagged with custom marketing variables, so that we can better record how people get to British Red Cross websites, and so better see what is generating the most traffic/conversions.

Jump to the actual link builder ⬇⬇

This link builder is designed with the following principles in mind:

  • It fits a taxonomy: the different types of marketing we do have been bucketed together into categories; this tool then matches those categories to marketing fields in other tools (Google Analytics, Fundraising's web reporting database for donate.redcross.org.uk)
  • It is restrictive: there are a lot more dropdowns than free text boxes, to enforce consistency across the organisation and adherence to the taxonomy.
  • It is selective: each user path that this tool can create links for has been scoped out, prioritised, and fit into our tracking taxonomy.

What is the scope of the tool?

This link builder is designed to cover the standard user paths people take to access our site. As it is restrictive and selective, there may be activity you want to track that isn't possible to create a link for with this tool.

If something falls outside the scope of this tool, then the options are:

  • tracking leads can create ad-hoc tracked links that still matches our tracking taxonomy. As this is a manual process, it's not something we offer for general tracking requests as standard, but may consider for high-value activity (e.g. tracking activity for a celebrity or corporate partner).
  • tracking leads can update the link builder to accommodate the new path. This will involve requirements gathering, user story generation, etc. and so is not a 'quick fix'.

Use notes

  • This tool is only for links going to a BRC site page. If you are linking to another platform that we don't own (e.g. Youtube) then there is no point using this tool.
  • Use this tool with the full URL for a page (i.e. don't create tracked links that point to an alias)
  • "Individual-level tracking" is extra tracking we add, so that systems capturing personal data (e.g. the donate platform) can also record this marketing tracking information too.
  • Source codes are a tool that fundraising uses to describe activity on the database Ascent. Online source codes are "approach codes" that aren't linked to a given fund (unlike much of the offline fundraising set-up).
